Screen & Element Capture Web Component Demo

This demo showcases two ways to capture content:

  • Capture Screen: Uses the browser's native Screen Capture API (`getDisplayMedia`) to let you select a screen, window, or tab.
  • Capture Specific Element: Uses the experimental Element Capture API (`element.captureStream()`) to capture a video stream of a specific DOM element. The element must have an ID.
